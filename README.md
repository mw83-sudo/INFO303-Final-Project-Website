# INFO303-Final-Project-Website

This repo is a **static website** you can run locally or deploy to GitHub Pages.

## Run locally
```bash
python3 -m http.server 8000
```
Open: http://localhost:8000

## Rebuild the micro video
```bash
bash scripts/build_micro_video.sh
```

## Replace placeholders
- Put your Project 6 PDF at `assets/docs/project6-qna.pdf`
- Put your sister vlog at `assets/video/sister-vlog.mp4` (or embed a YouTube unlisted link)
